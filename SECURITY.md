# Security Policy

## Supported versions

The latest release on PyPI receives security fixes. `django-bikram-sambat` is a pure
date library: the core has no dependencies and performs no network or filesystem
access at runtime, so its attack surface is small.

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue.

- **Preferred:** GitHub's private vulnerability reporting. Go to the repository's
  **Security** tab → **Report a vulnerability**.
  _(Maintainer: enable this once under Settings → Code security → Private
  vulnerability reporting.)_
- **Or:** email <anishbhimgc@gmail.com>.

You'll get an acknowledgement within a few days. Please allow reasonable time for
a fix before any public disclosure.
